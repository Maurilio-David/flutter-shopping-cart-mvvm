# Aplicativo Flutter de carrinho de compras desenvolvido como desafio técnico, aplicando arquitetura MVVM, ChangeNotifier, Result Pattern e boas práticas de separação de responsabilidades.

## Tecnologias
* Flutter 3.35.7
* Dart
* Provider
* Dio
* ChangeNotifier
* flutter_test

## Como executar

- flutter pub get
- flutter run

## Teste

- flutter test
